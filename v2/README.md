# ZEMEK-CV / v2

Tato složka obsahuje podpůrné soubory pro novou verzi životopisu. Veřejný vstupní soubor `index.html` zůstává o úroveň výš v kořeni `ZEMEK-CV`.

## Obsah

- `cv-cs-hr.md` - hlavní finální verze životopisu pro český trh
- `positioning-notes.md` - vysvětlení role framingu a volby titulku
- `README.md` - stručný popis obsahu složky a doporučené použití
- `styles.css` - vzhled webové verze
- `script.js` - interakce webové verze
- `avatar-2026.png` - avatar použitý ve webové verzi

## Použití

- `cv-cs-hr.md` je základní verze pro HR screening a běžné posílání na pozice.
- `positioning-notes.md` slouží jako opora, pokud bude potřeba životopis dále upravovat pro konkrétní typ role.
- `../index.html` otevřete přímo v prohlížeči jako statickou stránku bez build kroku.

## Poznámka

Tato verze je postavená tak, aby:

- byla čitelná pro první HR screening
- zachovala silnou C# / .NET identitu
- současně ukázala applied AI praxi a solution přesah
